Ce fichier vous explique comment installer et lancer le 2048 à partir du fichier fourni sur Windows.


Pour pouvoir jouer au 2048, vous devez obligatoirement utiliser le fichier "2048.exe" fourni. Il vous suffit de lancer l'exécutable où vous voulez.
Pensez à autoriser le fichier par votre antivirus pour éviter tout malfonctionnement du programme.


Lors d'une partie, le programme créera deux fichiers .txt qui ont pour but de sauvegarder les meilleurs scores et votre partie en cours.
Si vous souhaitez déplacer le programme dans un autre dossier ou via une clé USB par exemple, il faudra déplacer également les deux fichiers .txt afin de garder les meilleurs scores et votre sauvegarde si vous n'avez pas terminé votre partie.
